Learning R with GenAI


https://www.tidyverse.org/blog/2025/04/learn-tidyverse-ai/

Highlights

  • ChatGPT will give you code.
  • It may or may not work.
  • It may or may not be good code (probably not).
  • You should always cross check it with the documentation.


Suggestions for Using ChatGPT to Learn R

  • “Use code completion tools sparingly if you’re a new user.”
  • ChatGPT is good for getting starter code, discovering functions, arguments, solving a simple, well-defined problem, etc.
  • Break a task into smaller steps.
  • Don’t depend on ChatGPT to create a workflow or chain
  • Be wary of code you don’t understand.
  • R has great documentation & tutorials online - take advantage of it.


Other Tips

  • Claude is good at code
  • Give models the ability to read R package documentation using the Model Context Protocol (source)
  • Some people like the GenAI code-completion features in IDEs
